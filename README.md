# K-Nearest Neighbors (KNN) From Scratch

This is a simple implementation of the **K-Nearest Neighbors (KNN) algorithm** in Python, built **from scratch** without using machine learning libraries like scikit-learn.  

It uses the **Iris dataset** to demonstrate classification using KNN.

- Fully **beginner-friendly** code
